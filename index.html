Summary of Project: Twitch API Metrics

The purpose of this project was to provide insight to potential new streamers and game developers on how the different metrics that define streamers/games' influence on Twitch.tv changes over time and where there might be the most room for market development. Data was compiled every hour after making JSON requests to the Twitch API. Due to the scope and timeline of the project, data was collected over the course of one week.

After accumulating the data, data manipulation and data frame creation was done using Pandas. Next, Matplotlib was used to plot available data such as Total Views for the Top Games endpoint, Views over Time, Streamers over Time, etc. The site provides the source data and visualizations created as part of this analysis, as well as conclusions and insights that were derived from this analysis.
